# Mi-Ruta-C-plus-plus

> **Objetivo:** Aprender a programar en C++ construyendo lógica desde cero, **sin copiar código**.

---

# 🎯 Objetivo Final

Al terminar estas 8 semanas deberías ser capaz de crear por tu cuenta:

- ✅ Menús interactivos
- ✅ Sistemas sencillos
- ✅ Programas de **100 a 300 líneas**
- ✅ Uso de funciones
- ✅ Uso de arreglos
- ✅ Uso básico de `struct`

> **La meta no es memorizar código, sino aprender a resolver problemas.**

- `int`

- `double`

```cpp

/
%
```

### Comparación

```cpp

!=
<
>
<=
>=
```

### Condicionales

```cpp

```

# 🗓 Semana 4 · Funciones

## 📖 Teoría

Funciones que:

- No reciben parámetros
- Reciben parámetros
- Devuelven valores

Ejemplo:

```cpp
int sumar(int a, int b)
{
    return a + b;
}
```

## 💻 Ejercicios

- [ ] Función sumar
- [ ] Función promedio
- [ ] Función `esPar`
- [ ] Función `mayorDeTres`
- [ ] Calculadora usando funciones

### 🎯 Meta

Dividir programas grandes en pequeñas funciones.

# 🧠 La Herramienta Secreta

Antes de abrir Visual Studio Code (o cualquier IDE), responde estas tres preguntas:

## 📥 ENTRADA

¿Qué datos me proporciona el usuario?

## ⚙️ PROCESO

¿Qué cálculos o lógica necesito realizar?

## 📤 SALIDA

¿Qué información mostraré?

---

## 💡 Ejemplo

### ENTRADA

```text
Nombre
Herederos
Fortuna
```

### PROCESO

```text
Calcular sobrante
Dividir la herencia
Calcular impuestos
```

### SALIDA

```text
Monto por heredero
Impuesto
Cantidad destinada a caridad
```
